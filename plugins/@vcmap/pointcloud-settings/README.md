# @vcmap/pointcloud-settings

> Part of the [VC Map Project](https://github.com/virtualcitySYSTEMS/map-ui)

The **Pointcloud Settings** plugin adds an action to point cloud layers that lets you modify their parameters. This action is accessible in the overflow of each pointcloud layer, in the content tree.

This plugin has no configuration. Once added to the Map, all pointcloud layers will receive the additional action.
