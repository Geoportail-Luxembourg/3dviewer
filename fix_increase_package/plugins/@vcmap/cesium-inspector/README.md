# @vcmap/cesium-inspector

A plugin for adding Cesium debugging inspectors to the VC Map:

- [CesiumInspector](https://cesium.com/learn/cesiumjs/ref-doc/CesiumInspector.html)
- [Cesium3DTilesInspector](https://cesium.com/learn/cesiumjs/ref-doc/Cesium3DTilesInspector.html)
- [VoxelInspector](https://cesium.com/learn/cesiumjs/ref-doc/VoxelInspector.html)

A tool button is added to the menu dropdown within the navbar.

The tool is activated on startup, if search param `debug` is set to true, e.g.:
http://localhost:8008/?debug=true
